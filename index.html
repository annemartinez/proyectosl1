Etiquetas de Tecnologias Utilizadas


Etiquetas CSS

Nombre ¿Para qué sirve?
Color Para el color del texto
Letter-spacing Para el espacio entre caracteres
Text-align Define la alineación del texto
Color 12.99
Border-bottom Para el borde inferior

Etiquetas HTML

Nombre ¿Para qué sirve?
body Para el contenido de la página.
head Para información sobre el documento.
a Para colocar los enlaces dentro de la pagina.
div Genera una división dentro del contenido.
br Para hacer saltos de linea.

Etiquetas ViewPort

Nombre ¿Para qué sirve?
Width Para la anchura virtual (emulada) de la pantalla o anchura del viewport.
Height Para la altura virtual de la pantalla o anchura del viewport.
Initial-scale Para la escala inicial del documento.
Minimum-scale: Determinar la escala mínima que se puede poner en el documento.
User-scalable Define si se permite o no al usuario hacer zoom.
